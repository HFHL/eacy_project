# 曦栋智能CRF数据平台 - 高保真交互原型

## 项目概述

本项目是曦栋智能CRF数据平台的高保真交互原型，基于React + Ant Design 5.x构建，完整实现了PRD文档中定义的五大核心模块功能。

## 核心架构理念

### 患者数据池 + 专项研究双层架构
- **患者数据池**：全集患者管理层，包含系统中所有患者档案
- **科研项目**：专项研究层，从患者数据池筛选患者进行深度研究

### 五大核心模块
1. **智能文档处理中心** - 多格式文档上传和AI智能处理
2. **CRF表单设计器** - 分组折叠式字段配置界面
3. **患者数据池管理** - 患者总览、筛选和批量操作
4. **科研数据集管理** - 项目管理、患者筛选、数据导出
5. **基础支持功能** - 用户系统、全局搜索、系统设置

## 技术栈

- **前端框架**: React 18+ + TypeScript
- **UI组件库**: Ant Design 5.x
- **状态管理**: Redux Toolkit
- **路由管理**: React Router v6
- **构建工具**: Vite
- **样式方案**: CSS Modules + Ant Design主题定制

## 设计规范

### 色彩规范
- **主色**: #1677ff (Ant Design Blue)
- **成功色**: #52c41a
- **警告色**: #faad14
- **错误色**: #ff4d4f
- **中性色**: rgba(0, 0, 0, 0.88)

### 医疗数据界面特殊规范
- **数据完整度**: 绿色(>90%) / 黄色(70-90%) / 红色(<70%)
- **置信度指示**: 高(绿色) / 中(黄色) / 低(红色)
- **患者信息**: 脱敏显示，保护隐私
- **大数据量**: 虚拟滚动优化

### 布局规范
- **栅格系统**: 24栅格
- **间距系统**: 8px倍数
- **响应式断点**: xs(480px), sm(576px), md(768px), lg(992px), xl(1200px), xxl(1600px)

## 用户流程

### 核心流程
1. **文档上传** → AI处理 → 患者匹配 → 数据归档
2. **患者数据池** → 筛选患者 → 科研项目 → 数据抽取
3. **CRF设计** → 模版配置 → AI抽取 → 数据导出

### 双层架构体现
- 患者数据池作为全集管理，所有患者统一管理
- 科研项目从患者数据池筛选，进行专项研究
- 数据在两层间同步，确保一致性
