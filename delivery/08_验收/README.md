# 08 验收

> 本模块面向**验收方**，提供验收范围、清单、性能/安全要点。

## 计划覆盖

| 文档 | 内容 |
|---|---|
| **验收范围.md** | 本次交付的功能边界（什么在内、什么不在内） |
| **功能验收清单.base** | Bases 视图：聚合 `02_业务域` 下所有 `验收要点` |
| **性能验收.md** | 关键场景的性能指标与测试方法（如 100 页 PDF 抽取耗时） |
| **安全验收.md** | 鉴权、敏感数据处理、日志脱敏、依赖安全 |

## 验收原则

- 每个业务域的**验收要点**写在自己域内（`02_业务域/<域>/验收要点.md`），本模块**只做聚合**，不重复编写。
- 验收用例使用 [[T-验收用例]] 模板，关联到具体业务流程文档。

## 不在范围内

- 单元测试用例（看 `backend/tests/`）
- 自动化测试报告（看 CI 输出）
- 内部代码审查记录
