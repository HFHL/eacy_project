# 页面地图

> 路由 → 页面组件 → 主要后端 API 三栏对照。完整路由表参见源码（不再 copy），本文只记录**每条路由对应哪几个核心后端调用**，便于排查"点了某个按钮发了哪些请求"。

## 一、路由分组

```mermaid
graph LR
  Login[/login] --> Home[/dashboard]
  Home --> DocFlow[文档处理]
  Home --> Patient[患者数据池]
  Home --> Research[科研数据集]
  Home --> Admin[管理后台]
  DocFlow --> Upload[document/upload]
  DocFlow --> FileList[document/file-list]
  Patient --> Pool[patient/pool]
  Patient --> Detail[patient/detail/:patientId]
  Research --> Projects[research/projects]
  Research --> ProjectView[research/projects/:projectId]
  Research --> ProjectPatient[research/projects/:projectId/patients/:patientId]
  Research --> TemplateCreate[research/templates/create]
  Research --> TemplateEdit[research/templates/:templateId/edit]
  Research --> ProjectTemplate[research/projects/:projectId/template/edit]
```

## 二、对照表（顶层页面）

| 路由 | 页面组件 | 主要后端 API |
|---|---|---|
| `/login` | `pages/UserSystem/Login` | `POST /auth/login`、`POST /auth/refresh` |
| `/dashboard` | `pages/Dashboard` | `GET /stats/*`（见 `api/stats.js`） |
| `/document/upload` | `pages/DocumentUpload` | `POST /documents`（multipart 上传，可带 `patient_id` 直接归档） |
| `/document/file-list` | `pages/FileList` | `GET /documents`、`GET /documents/v2/tree`、`GET /documents/v2/groups/:id/documents`、`POST /documents/v2/groups/:id/confirm-archive`、`POST /documents/batch-archive` |
| `/document/processing` | `pages/AIProcessing`（隐藏菜单） | `GET /documents`、`PATCH /documents/:id`、`POST /documents/:id/archive` |
| `/patient/pool` | `pages/PatientPool` | `GET /patients`、`POST /patients`、`DELETE /patients/:id`、`POST /patients/ehr-extraction-status` |
| `/patient/detail/:patientId` | `pages/PatientDetail` | `GET /patients/:id`、`GET /patients/:id/ehr`、`GET /documents?patient_id=...`、`POST /extraction-jobs`、`POST /patients/:id/ehr/update-folder`、`PATCH /patients/:id/ehr/fields/:fieldPath`、`GET /extraction-jobs/:id`、`GET /task-batches/:id`、`GET /patients/:id/ehr/fields/:fieldPath/{events,evidence,candidates}` |
| `/research/projects` | `pages/ResearchDataset` | `GET /projects`、`POST /projects`、`PATCH /projects/:id`、`DELETE /projects/:id`、`GET /schema-templates`、`POST /schema-templates`、`DELETE /schema-templates/:id` |
| `/research/projects/:projectId` | `pages/ResearchDataset/ProjectDatasetView` | `GET /projects/:id`、`GET /projects/:id/patients`、`POST /projects/:id/patients`、`DELETE /projects/:id/patients/:ppId`、`POST /projects/:id/crf/update-folder`、`POST /projects/:id/export`、`GET /projects/:id/template-bindings`、`GET /schema-templates/:id` |
| `/research/projects/:projectId/patients/:patientId` | `pages/ResearchDataset/ProjectPatientDetail` | `GET /projects/:id/patients/:ppId/crf`、`PATCH /projects/:id/patients/:ppId/crf/fields/:path`、`POST /projects/:id/patients/:ppId/crf/update-folder`、`POST /projects/:id/patients/:ppId/crf/records`、`GET /projects/:id/patients/:ppId/crf/fields/:path/{events,candidates}` |
| `/research/projects/:projectId/template/edit` | `pages/ResearchDataset/ProjectTemplateDesigner` | `GET /projects/:id/template-bindings`、`GET /schema-templates/:tid`、`POST /schema-templates/:tid/versions`、`POST /schema-template-versions/:vid/publish`、`POST /projects/:id/template-bindings` |
| `/research/templates/create` | `pages/CRFDesigner` | `POST /schema-templates`（含 designer + schema_json） |
| `/research/templates/:templateId/edit` | `pages/CRFDesigner` | `GET /schema-templates/:id`、`POST /schema-templates/:id/versions`、`POST /schema-template-versions/:vid/publish`、`PATCH /schema-templates/:id` |
| `/research/templates/:templateId/view` | `pages/CRFDesigner` (只读) | `GET /schema-templates/:id` |
| `/admin` | `pages/Admin` | `GET /admin/{users,projects,templates,documents,stats,extraction-tasks}`、`GET /admin/extraction-tasks/:id`、`GET /admin/extraction-tasks/:id/events` |
| `/user/profile` | `pages/UserSystem/UserProfile` | `GET /auth/me`、`PATCH /auth/me` |
| `/user/settings` | `pages/UserSystem/SystemSettings` | `GET /auth/settings`、`PATCH /auth/settings` |

## 三、布局与守卫

- 除 `/login` 外所有路由统一挂在 `MainLayout` 下（见 [[组件复用说明#MainLayout]]）。
- 鉴权：未带 `access_token` 触发 401 时由 `api/request.js` 自动 redirect 到 `/login`，并尝试一次 `POST /auth/refresh`。
- 顶部导航菜单不是路由树自动生成，而是 `router/index.jsx` 末尾 `routeConfig.mainMenu` 手动配置。
